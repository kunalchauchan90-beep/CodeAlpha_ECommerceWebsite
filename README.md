# CodeAlpha E-Commerce Website

## Project Description
A simple Django-based E-Commerce Website developed as part of the CodeAlpha Full Stack Development Internship.

## Features
- User Registration
- User Login & Logout
- Product Listing
- Product Details
- Add to Cart
- Place Order
- Django Admin Panel

## Technologies Used
- Python
- Django
- HTML
- CSS
- SQLite

## Run Project

```bash
python manage.py runserver
```

Open:

http://127.0.0.1:8000/
